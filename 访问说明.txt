Streamlit OCR 辨识工具 - 访问说明
=====================================

程序已启动，请通过以下方式访问：

1. 在浏览器中打开以下任一地址：
   - http://localhost:8501
   - http://127.0.0.1:8501

2. 如果无法访问，请尝试：
   a) 检查防火墙设置，允许 Python 和 Streamlit 通过防火墙
   b) 关闭其他可能占用 8501 端口的程序
   c) 使用管理员权限运行程序

3. 手动启动程序：
   - 双击 "启动程序.bat" 文件
   - 或在命令行中运行：streamlit run app.py

4. 如果端口被占用，可以指定其他端口：
   streamlit run app.py --server.port 8502

注意事项：
- 程序启动后会自动在浏览器中打开
- 如果浏览器没有自动打开，请手动访问上述地址
- API Key 已预设，可以直接使用
