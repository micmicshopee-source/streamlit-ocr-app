## ⚠️ 如果遇到權限問題

### 解決方法
1. **關閉可能占用 Git 的程序**
   - 關閉 VS Code
   - 關閉 Git GUI 工具
   - 關閉其他可能使用 Git 的程序

2. **以管理員身份運行 PowerShell**
   - 右鍵點擊 PowerShell
   - 選擇「以系統管理員身分執行」
   - 重新執行上述命令

3. **檢查文件權限**
   - 確保對 `.git` 目錄有寫入權限
   - 檢查防病毒軟件是否阻止 Git 操作
